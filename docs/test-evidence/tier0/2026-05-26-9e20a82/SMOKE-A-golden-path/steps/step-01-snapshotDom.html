bonsAI - Beta Notice
Welcome to bonsAI! This plugin is currently in beta. Some features may not work as expected, and AI-generated recommendations — especially TDP and performance changes — should be verified before relying on them. bonsAI modifies system hardware settings based on AI suggestions. Use at your own risk. If you have another PC on your LAN that can host Ollama, that path is typically much faster than inference on-device. Running heavy local AI while a game has high VRAM / graphics load may crash the game or cause unstable behavior from memory pressure — use at your own risk. To report bugs or request features, visit: https://github.com/cantcurecancer/bonsAI/issues By continuing, you acknowledge this is experimental software.
